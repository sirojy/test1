## Git 기초 명령어

## git init 
로컬 저장소 생성

## git add <파일명> 
특정 파일/ 폴더의 변경사항 추가

## git commit -m '<커밋메시지>'
커밋(버전기록)

## git status 
상태 확인
<br>
## git log
버전 확인
<br>
<br>
